Number & format conventions

Symbol + abbreviate at ≥1M (K/M/B). Full value £1,490,000 only in tables.

Ratios to 2 dp with x; percentages to 1 dp. No space before % or x.

Signed with a triangle; Fern up, Ember down. Always relative to a stated baseline.

Day-month-year, month abbreviated, no ordinals. Ranges: 1–30 Jun.
ALIGNMENT

Tabular figures, right-aligned in tables. Thousands separators always.
ZERO & NULL
0  ·  —
Real zero is 0; no data is an em-dash in Warm Grey. Never blank, never 0 for missing.

Charts at small size & in print

Thumbnails, deck tiles, and PDF exports: drop gridlines and axis ticks, keep one direct label per series, and raise line weight to ≥2.5px so Coral still leads. Never rely on hover — bake the key numbers into the chart. Export chart SVGs as raster (data-om-raster) so they survive PPTX and PDF unchanged.

Shared behaviour

ONE SHADOW · ONE SCRIM · ONE ESC
ELEVATION

Every overlay uses shadow-md and a hairline edge — never a heavier shadow, never a shadow plus a thick border.

SCRIM

Modals and drawers only — Ink at ~44%. Menus, popovers, and tooltips never dim the page.

FOCUS

Trap focus in modals and drawers; return it to the trigger on close. Aqua focus ring on dimmed dark ground.

MOTION

Fade + 8px rise (drawers slide from the edge), 180ms ease-out. Esc and outside-click close everything. Honour reduced-motion.

05 — Responsive & gridDESIGN.md ↗

One system, every width.

The same tokens hold from a phone to a wide monitor. A 12-column grid collapses in steps, type and section padding flex with clamp(), and the app shell rearranges — never redraws.

Content is capped at 1200px and centred. Columns halve in steps rather than reflowing arbitrarily — 12 → 8 → 4.

The shell rearranges, never redraws

Full sidebar on desktop, a 60px icon rail on tablet, and on mobile the nav moves to a bottom tab bar with the top bar holding the menu and title. Same order, same active Coral signal, everywhere.

Touch before hover

On small screens every target is ≥ 44px and nothing hides behind hover — tooltips and hover menus get a tap equivalent.

Stack, don't shrink

Multi-column layouts collapse to a single column in priority order rather than squeezing. Cards keep their padding and radius.

Keep the action reachable

The one primary action stays in reach — a sticky bottom bar on mobile, top-right on desktop. Modals become full-height sheets under 640px.
